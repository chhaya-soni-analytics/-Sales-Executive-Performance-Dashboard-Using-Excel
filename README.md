# 📊 Sales Executive Performance Dashboard (Excel)

## 🚀 Project Overview

This project focuses on analyzing sales performance using an interactive Excel dashboard. The goal is to transform raw sales data into meaningful insights that help evaluate **sales executives, regional performance, and target achievement**.

The dashboard enables business users to quickly identify performance gaps, top performers, and overall sales trends.

---

## 🎯 Objectives

* Evaluate **sales executive performance against targets**
* Identify **target gaps (Away From Target)**
* Analyze **region-wise sales contribution**
* Measure **target achievement efficiency**
* Track **daily sales trends**

---

## 📊 Dashboard Features

### 🔹 Key Metrics (KPIs)

* Total Sales
* Average Target Hit %
* Best Performing Region
* Performance Gap (Away From Target)

---

### 📈 Visualizations

* Sales Executive vs Total Sales (Bar Chart)
* Target Hit % Analysis
* Away From Target (Gap Analysis)
* Region-wise Total Sales
* Daily Sales Trend (Line Chart)

---

## 🎛️ Interactivity

The dashboard includes slicers for:

* Region
* Sales Executive

👉 Enables dynamic filtering and detailed performance analysis

---

## 🧹 Data Preparation

* Checked data consistency (names, regions, values)
* Removed duplicates and handled missing values
* Converted dataset into Excel Table
* Created calculated fields for:

  * Total Sales
  * Target Hit %
  * Away From Target

---

## 🛠️ Methodology

### ✔ Step 1: Data Understanding

Analyzed dataset structure (sales, target, region, daily data)

### ✔ Step 2: Data Cleaning

Ensured data accuracy and consistency

### ✔ Step 3: Feature Engineering

Created calculated fields:

```excel id="ex1"
Target Hit % = (Total Sales / Target) * 100
```

```excel id="ex2"
Away From Target = Target - Total Sales
```
